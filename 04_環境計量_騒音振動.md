# ファイル04：騒音・振動調査
## 廃棄物施設の騒音振動・自動車騒音常時監視

> **適用業務例**：令和7年度騒音及び振動調査委託、令和7年度自動車騒音常時監視業務委託  
> **関連法令**：騒音規制法、振動規制法、JIS Z8731、JIS Z8735

---

## 1. サブタイプの区別

| サブタイプ | 業務概要 | 積算の特徴 |
|----------|---------|-----------|
| **A: 廃棄物施設騒音振動** | 廃棄物処理施設の敷地境界での騒音・振動測定 | 施設数×時間帯×測定点数。パッカー車対応が特殊 |
| **B: 自動車騒音常時監視** | 面的評価支援システムを用いた沿道評価 | 評価区間数・測定地点数・GISデータ作成が主コスト |

---

## 2. サブタイプA：廃棄物施設騒音振動

### 抽出場所

| 抽出項目 | 記載箇所 |
|---------|---------|
| 対象施設名・住所 | 「履行場所」セクション |
| 測定点数 | 「別表1 測定点数一覧」（施設×時間帯×敷地境界/補助地点） |
| 測定種別 | 「調査項目」セクション（騒音①・騒音②・振動） |
| 時間帯区分 | 朝（6-8時）・昼間（8-19時）・夕（19-23時）・夜間（23-翌6時） |
| 測定間隔 | 「測定開始時刻は○時間以上間隔を空けて」の条件 |
| パッカー車の特殊測定 | 「騒音調査②」の有無と時間帯（深夜1時〜5時等） |

### 測定点数の計算

```
施設ごとの年間測定点数 =
  騒音①: Σ(時間帯×回数×箇所数) + 補助地点
  騒音②: 回数 × 1箇所
  振動: Σ(時間帯×回数×箇所数)

例（廃棄物施設3施設の場合）:
  東部・西部: 騒音①20点+騒音②5点+振動12点 = 各37点
  臨海: 騒音①20点+騒音②5点+振動4点 = 29点
  合計: 103点/年
```

> ⚠️ **補助地点**は敷地境界とは別に設定される。別表1で「-」と「数値」の区別を確認する。

## 3. サブタイプB：自動車騒音常時監視

### 抽出場所

| 抽出項目 | 記載箇所 |
|---------|---------|
| 評価区間数・延長 | 本文「(1)自動車騒音面的評価」冒頭 |
| 定点・年度調査区間の内訳 | 別紙1-1「実施計画」 |
| 騒音測定地点数 | 「(2)沿道騒音レベルの測定」の「実測区間（○地点）」 |
| 振動測定地点数 | 「(3)沿道振動レベルの測定」の「定点実測区間（○区間）」 |
| 交通量調査地点数 | 「(4)交通量調査」の調査地点数 |
| システム・PC貸与 | 「面的評価支援システム」「ノートパソコンは本市から貸与」 |

### 成果品の特殊性

自動車騒音常時監視は**成果品の種類が多い**。全て確認すること。

```
□ 調査報告書（A4版 簡易製本）: 2部
□ 報告書の概要版（A4版 カラー 40ページ程度）: 10部
□ 騒音・振動測定の調査報告書（速報）: 1部（電子データ可）
□ 環境省報告用資料
    □ 常時監視フォーマット（住居戸数0含む・除くの両パターン）
    □ 環境GISフォーマット
    □ 騒音測定箇所の位置図・詳細図
    □ 実施計画
□ 報告書電子データ（PDF・Excel）: 1式
□ オブジェクト・データベース: 1式
□ 計量証明書（騒音・振動）: 添付
```

---

## 4. 報告書の提出時期

| 成果物 | 提出時期 |
|--------|---------|
| 速報（騒音・振動） | 騒音測定終了後（原則11月末までに測定完了） |
| 最終報告書 | 年度末（3月） |
| 年間報告書（廃棄物施設） | 採取終了後2ヶ月以内（毎回）+ 年間1部（電子） |
