# ファイル05：大気測定（PM2.5サンプリング）

> **適用業務例**：令和8年度微小粒子状物質サンプリング業務委託  
> **関連法令**：大気中微小粒子状物質（PM2.5）成分測定マニュアル（環境省、令和元年5月改訂）

---

## 1. このカテゴリの業務特性

- **試料採取のみ**が業務範囲（分析は別業者）
- 装置の搬入・設置・保守・撤去・フィルタ交換が主作業
- 季節ごとに**コア期間**が設定され、コア期間は変更不可
- フィルタ・フィルタケース・ポリ袋は**本市支給**（積算に含めない）

---

## 2. 抽出場所リスト

| 抽出項目 | 記載箇所 |
|---------|---------|
| 採取地点数・場所 | 「(2)実施場所」 |
| 季節ごとの採取期間 | 「(3)調査期間（試料捕集期間）」の表 |
| コア期間 | 調査期間表の「コア期間」行 |
| 採取試料数（種別別） | 「(4)採取試料数」の表（成分分析用・二重測定用・トラベルブランク用） |
| 装置・フィルタの支給/自前区分 | 「(6)試料採取用の装置等」 |
| 報告（速報・報告書） | 「(9)報告」 |

---

## 3. 試料数の計算

```
試料数の構成（季節ごと）:
  春吉測定局: 成分分析用14 + トラベルブランク1 = 15枚
  元岡測定局: 成分分析用14 + 二重測定用2 + トラベルブランク1 = 17枚
  1季あたり合計: 32枚
  フィルタ種別: PTFEフィルタ・石英繊維フィルタの2種類 → 各局・各季で2倍

4季合計:
  (15 + 17) × 2種 × 4季 = 256枚

注意: フィルタは本市支給のため積算に含めない
     装置・その他資材は受託者が準備
```

---

## 4. 成果物要件

| 成果物 | 提出時期 | 形式 |
|--------|---------|------|
| 速報（各季） | 採取完了後30日以内 | 電子データ |
| 報告書（全4季） | 冬季採取完了後 | 電子データ |
| 実施計画書・標準作業書 | 春季採取開始前 | 要承認 |
| 試料採取概要説明資料 | 春季採取開始前 | A4 1枚程度 |
| 借用書（鍵使用時） | 事前 | 借用予定期間明記 |

# ファイル06：ごみ質調査・アスベスト分析

---

## A. ごみ質調査

> **適用業務例**：令和8年度西部工場ごみ質等調査委託  
> **関連法令**：廃棄物処理法第8条の3、昭和52年厚生省通知環整第95号

### 調査項目と頻度

```
生ごみ組成調査（8分類）: 7回/年（指定月）
単位容積重量: 同上
乾燥ごみ組成調査（9分類）: 7回/年
水分・真灰分・水素分・発熱量: 7回/年
焼却残渣 水分・大型不燃物割合: 11回/年（10月除く毎月）、毎月2炉調査

注意: 施設の稼働状況により焼却残渣の回数が変更される可能性あり
```

### 抽出場所リスト

| 抽出項目 | 記載箇所 |
|---------|---------|
| 調査項目・回数 | 「5 調査方法」の表 |
| 指定月 | 表の「調査回数」列（括弧書きで月を指定） |
| 試料採取手順 | 「(2)試料の採取及び分析手順」 |
| 焼却残渣の引き渡し条件 | 「④焼却残渣の水分及び大型不燃物割合」㋒の記述 |
| 調査予定表の提出期限 | 「6 調査日」（契約後10日以内） |

### 成果物要件

```
毎回の報告書: 採取終了後1ヶ月以内に電子データで提出
  - 本市指定のデータ処理ファイル
  - 写真（A4版PDF: 採取直後・作業中・仕分け後・乾燥前後）
  - 焼却残渣: 任意様式可

年間報告書: 電子ファイル1部（PDF・Excel、しおり付き）
```

## B. アスベスト定性分析

> **適用業務例**：碓井野球場・他アスベスト含有分析調査業務委託  
> **関連法令**：JIS A 1481-1、JIS A 1481-2

### 特徴

- 建材・仕上げ塗材（下地調整塗材含む）の**定性分析**
- 1検体3箇所採取が標準
- **資格要件**が厳格（A/Bランク認定分析技術者、JEMCAインストラクター等）
- 含有確認時は**速報連絡**が必要

### 抽出場所リスト

| 抽出項目 | 記載箇所 |
|---------|---------|
| 調査施設・住所・建設年度 | 「調査場所」「住所」「建設年度」の表 |
| 調査箇所・検体数 | 「調査箇所」「検体数」の表 |
| 分析方法・規格番号 | 「業務委託概要」の「定性分析（JIS A 1481-1, -2）」 |
| 分析者の資格要件 | 「業務委託概要」の箇条書き3種 |
| 安全対策 | 「試料採取時の安全対策」セクション |
| 報告書 | 「報告書等」セクション |

### 成果物要件

```
報告書: 正本1部
内容: 採取者・分析者の資格証明 + 採取箇所図 + 写真
  写真: 着工前・完了後（左右対称）、飛散防止対策状況、保護具着用状況

速報義務: アスベスト含有確認時は「その検体数及び箇所を速報で通知」
```
