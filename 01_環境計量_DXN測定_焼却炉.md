# ファイル01：ダイオキシン類測定（焼却炉系）
## 排ガス・焼却灰・飛灰・排水・作業環境測定

> **適用業務例**：家畜保健衛生所廃棄物焼却施設ダイオキシン類測定、焼却工場ダイオキシン類等測定  
> **関連法令**：ダイオキシン類対策特別措置法（D法）、労働安全衛生規則第592条の2

---

## 1. このカテゴリの業務特性

焼却炉系DXN測定には**2つのサブタイプ**が存在する。

| サブタイプ | 主測定対象 | 主な根拠法令 | 積算の特徴 |
|----------|----------|------------|-----------|
| **A: 排ガス系** | 排ガス中DXN、ばい煙（SOx・ばいじん・HCl）、CO、O₂ | D法第28条、大気汚染防止法 | 現地サンプリングが高コスト。炉ごと×媒体×頻度が複雑 |
| **B: 排ガス以外** | 排水・焼却灰・飛灰・ばいじん・溶融メタル・溶融スラグ・汚泥、作業環境 | D法第28条、労安衛則 | 試料引取り・採取容器納入が業務に含まれる場合あり |

> **注意**：1つの仕様書にAとBが混在する場合（A=別業者、B=本業者）があるため、タイトルに「排ガス以外」の記載があるか必ず確認する。

## 4. 数量抽出の重要ポイント

### 4.1 施設ごとの媒体除外条件

飛灰（フライアッシュ）は**施設によって測定対象外**になることがある。
抽出時は必ず括弧書きの除外注記を確認し、`excluded=TRUE`を設定する。

```
確認手順:
1. 「調査対象媒体」セクションで媒体を列挙
2. 「ただし〜は除く」の括弧書きを探す
3. 除外施設名をexclusion_reasonに記録
4. 検体数を修正（除外施設分を差し引く）
```

### 4.2 作業環境測定の点数計算

作業環境測定は測定箇所（部屋）×測定種別（A測定1日目、A測定2日目、B測定、DXN粒子状、DXN ガス状）の組み合わせで計算する。

```
例（焼却工場DXN排ガス以外の仕様書より）:
  新門司工場: 3箇所（灰固化室・集塵機ヤード・溶融炉前）
  皇后崎工場: 2箇所（灰固化室・集塵機ヤード）
  測定種別: A測定(1日目/2日目)、B測定(1日目/2日目)、
           併行測定(1日目/2日目)、DXN粒子状(1日目)、DXN ガス状(1日目)
           = 8種別×各2点 = 16点/箇所
  総計: 5箇所×16点 = 80点（上半期・下半期で各1回 = 160点/年）

注意: 上半期・下半期で施設が異なる場合がある
  （例: 新門司工場=6月・12月, 皇后崎工場=8月・2月）
```

### 4.3 試料引取りコストの扱い

「採取容器の洗浄・納入」「試料の引取り」が業務範囲に含まれる場合は、
分析費用とは別に**現地作業費・交通費・容器費**を積算に含めること。

## 5. 成果物の特殊要件

### 焼却炉系DXNに特有の提出物

| 提出物 | 提出タイミング | 様式 |
|--------|-------------|------|
| 業務計画書（年間採取スケジュール含む） | 契約後・速やかに | 任意（測定方法・使用機器記載） |
| 作業連絡票 | 作業発生月の前月末まで | 指定様式 |
| 速報値（電子データ） | 試験結果が得られ次第 | 指定様式（報告様式1・2等） |
| 結果報告書（毎回） | 採取/引取り月の翌月末日 | 1部 |
| ダイオキシン類測定結果報告書 | 指定期日まで（例: 令和8年1月末） | 電子データ（報告様式3等） |
| 業務完了報告書 | 業務完了時 | 1部 |

> ⚠️ **「全ての業務及び報告を○月末日までに完了させる」** という期限が設定される場合がある（業務終了日≠履行期間終了日）。必ず確認すること。

## 6. 比較ロジック（年度間）

### 変更検知の重点チェック項目

```yaml
施設の変更:
  - 施設の増減（新設・廃止）
  - 炉の増減（A炉/B炉など）
  - 測定箇所（作業環境の部屋・面積の変更）

媒体・項目の変更:
  - 飛灰の除外施設の変更
  - 含水率測定の追加・削除
  - 作業環境測定の測定種別の変更
  - 新規媒体の追加（例: 溶融スラグが追加）

頻度・タイミングの変更:
  - 上半期・下半期の測定月の変更
  - 作業環境測定の回数（上半期/下半期 → 年1回等）

報告要件の変更:
  - 速報提出の追加
  - 報告様式の変更（指定様式番号の変更）
  - 業務完了期限の変更（例: 1月末 → 3月末）
```
