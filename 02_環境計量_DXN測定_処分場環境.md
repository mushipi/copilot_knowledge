# ファイル02：ダイオキシン類測定（処分場・環境系）
## 放流水・地下水・土壌・河川水のDXN測定

> **適用業務例**：嘉麻クリーンセンターダイオキシン類等調査測定業務（特記仕様書）  
> **関連法令**：ダイオキシン類対策特別措置法第7条、廃棄物最終処分場の維持管理基準省令

---

## 1. このカテゴリの業務特性

- 処分場周辺の**環境汚染モニタリング**が目的
- 土壌・地下水・河川水・放流水など**複数媒体**を同一仕様書で扱う
- DXN測定と**一般水質検査**が同一仕様書に混在することが多い（→ ファイル03「水質分析」との境界注意）
- 測定箇所が地区名・井戸番号で指定され、**「後日指定」**が多い

---

## 2. 抽出場所リスト

| 抽出項目 | 主な記載箇所 |
|---------|------------|
| 測定場所・検体数 | 別紙「測定場所及び測定分析対象」の表 |
| DXN測定の対象物質 | 別紙「測定分析項目明細」（PCDDs・PCDFs・コプラナーPCB） |
| 水質検査の項目数 | 別紙「測定分析項目明細」の水質検査セクション（項目番号を通し番号で数える） |
| 地区・井戸の指定状況 | 末尾の別紙マップ・測定箇所図（「後日指定」の有無） |
| 調査時期 | 本文またはスケジュール表（「1月実施予定」等） |
| 報告書要件 | 「報告」「守秘義務」セクション |

---

## 3. 検体数カウントの注意点

処分場環境系は**媒体×地点×回数**の三重構造になる。

```
例（嘉麻クリーンセンター特記仕様書より）:
  土壌: 3検体（①敷地内・②上区・③百々谷）
  放流水: 2検体（①浸出液処理水・②下流井戸）
  河川水: 1検体
  地下水（DXN+水質）: 3+1+1=5検体
  合計: 11検体

注意: 「後日指定」箇所は検体数=確定済みだが場所未定。積算では検体数を確定値として使用。
```

---

## 4. DXN＋水質の混在業務の分離ルール

1つの仕様書にDXNと一般水質が混在する場合、**費用計算上は分離**して積算する。

```
分離基準:
  DXN測定費用 = 検体数 × DXN単価（PCDDs/PCDFs/コプラナーPCB合計）
  水質検査費用 = 検体数 × 水質単価（項目数グループ別）
  
  ※ 同一検体でDXNと水質を同時測定する場合は、採取費用を按分または合算
```

## 5. 成果物要件

| 項目 | 内容 |
|------|------|
| 報告書 | A4版 2部（施設ごと等の指定がある場合あり） |
| 内容 | 測定分析結果・測定状況写真・採取箇所図面 |
| 守秘義務 | 結果等を第三者に漏らしてはならない |
| 再検査 | 目的達成のために必要な場合は受託者負担 |
