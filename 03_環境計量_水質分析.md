# ファイル03：水質分析・検査
## 浸出液処理水・地下水・河川水・保有水の水質検査

> **適用業務例**：環境保全に係る水質等分析業務、さくら浸出水処理施設水質検査業務  
> **関連法令**：廃棄物処理法施行規則第1条の7の4、環境基本法第16条、ダイオキシン類対策特別措置法（DXNが含まれる場合）

---

## 1. このカテゴリの業務特性

- 最終処分場の**維持管理モニタリング**が主目的
- 同一業務内に水質検査（1）・（2）・（3）など**複数の検査区分**が存在し、区分ごとに検査項目数・頻度・箇所数が異なる
- DXNは「水質検査の一部」として含まれる場合があり、ファイル02との違いは**一般水質がメイン**であること
- **頻度の混在**（月1回と年4回・年1回）が最も積算ミスを起こしやすい

## 3. 頻度混在の抽出ルール（最重要）

水質検査業務では**同一検査区分内でも項目グループごとに頻度が異なる**。
全項目を一律で計上するミスが最も多い。

### 頻度グループの分離手順

```
STEP 1: 「④検査回数」セクションを全文読み込む
STEP 2: 「〇〇項目については12回（毎月1回）」という記述を探す
STEP 3: 「その他の検査項目は4回」という記述を探す
STEP 4: 高頻度グループと低頻度グループに項目を振り分ける
STEP 5: 各グループの項目数×頻度×箇所数 = 年間検体数を計算
```

### さくら浸出水処理施設の例

| 採取場所 | 媒体 | 総項目数 | 高頻度グループ（月1回=12回/年） | 低頻度グループ（年1回） |
|---------|------|---------|------------------------------|-------------------|
| サンプリング場所 | 放流水 | 43項目 | 内6項目 | 内37項目 |
| 周縁観測井戸 | 地下水（上流） | 36項目 | 内7項目 | 内29項目 |
| 周縁観測井戸 | 地下水（下流） | 36項目 | 内7項目 | 内29項目 |

```
年間検体数の計算例（放流水）:
  高頻度: 6項目 × 12回 = 72検体相当
  低頻度: 37項目 × 1回 = 37検体相当
  → 単純に「43項目 × 12回」で積算してはいけない
```

---

## 4. 観測井戸の深度別扱い

下流側観測井戸が**深度別（30m・50m・70m）**に分かれる場合、各深度を別箇所として計上する。

```
例（環境保全水質分析業務）:
  下流側観測井戸（新1号井戸）: 30m・50m・70m の3箇所
  → 検査箇所数は「3箇所として計上」
  ただし、頻度が30m・50m・70mで統一かどうかを確認する
```

## 5. 検査方法の対応表

| 媒体・項目 | 検査方法 |
|----------|---------|
| 一般水質（最終処分場浸出液処理水） | 「一般廃棄物の最終処分場又は産業廃棄物の最終処分場に係る水質検査の方法」 |
| 地下水（環境基準項目一部） | 「地下水の水質汚濁に係る環境基準について」の測定方法 |
| 塩化物イオン | JIS K 0102-2 |
| 水素イオン濃度（pH） | JIS K 0102-12 |
| 飲料水基準項目 | 水質基準に関する省令の規定に基づき厚生労働大臣が定める方法 |
| ダイオキシン類（水質） | 「最終処分場に係るダイオキシン類の水質検査の方法」 |
